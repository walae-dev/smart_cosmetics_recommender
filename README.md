Smart Cosmetics Recommender

This project is a web-based cosmetic platform with a built-in recommendation system.

Features

* Product catalog (makeup, skincare, perfumes, boxes)
* Smart recommendation system based on:
    * Category
    * Skin type (for skincare)
    * Budget
* Dynamic filtering logic
* Interactive user interface

Technologies

* HTML
* CSS
* JavaScript

Project Structure

* products.js: product database
* script.js: logic and recommendation system
* HTML/CSS: interface
